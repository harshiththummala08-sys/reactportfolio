Thummala Harshith Reddy React Portfolio

Files:
- index.html
- package.json
- src/main.jsx
- src/styles.css
- Thummala-Harshith-Reddy-Resume.pdf
- assets/profile.jpg

Photo:
Put your photo inside the assets folder and name it profile.jpg.
The page already points to that file. Until the photo is added, it shows HR initials.

Run locally:
1. npm install
2. npm run dev

Netlify:
Upload this folder to GitHub, then connect the repository to Netlify.
Build command: npm run build
Publish directory: dist

GitHub:
Create your own repository and upload these files there.
Do not upload these files to your friend's repository.

GitHub Pages:
This project is configured for a repository named reactportfolio.
After uploading to GitHub, enable Pages with GitHub Actions.
The final link will be:
https://harshiththummala08-sys.github.io/reactportfolio/
